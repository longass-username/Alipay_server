------------Alipay Test Server------------
кодировка :UTF-8
язык      :Python 3.7

Программа была создана для тестирования 
Alipay операций.

На компьюетере должны быть:
- интерпретатор Python 3.X
- библиотека konfig

Описание работы с программой:
- Откройте файл config.ini и введите необ-
ходимые значения:
	-Адрес сервера(SERVER_ADDRESS)
	-Порт сервера(SERVER_PORT)
	-Вывод hex dump запроса/ответа
	(HEX_DUMP),может принимать значения
	0-выкл, 1-вкл

	-Раздел [Ali_settings] временно не
	работает, опции в этом разделе
	не имеет смысла менять
- Запустетие сервер (server.py)
- Откроется консоль и в нее будут выво-
диться получаемые и отправляемые даные
